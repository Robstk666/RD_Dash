# Инструкция по выгрузке и интеграции дизайна

Ты можешь перенести этот дизайн в свой проект несколькими способами:

### 1. Просмотр и копирование кода (HTML/CSS)
Над каждым экраном на канвасе есть кнопка **</> View Code**. 
- Нажми на неё, чтобы открыть панель с кодом.
- Ты можешь скопировать весь HTML и CSS (используется Tailwind CSS для стилизации).
- Этот код полностью адаптивен и готов к вставке в твой React/Vue/HTML проект.

### 2. Экспорт в Figma
Если тебе нужно доработать графику или передать макеты:
- В верхней панели инструментов нажми кнопку **Export to Figma**.
- Это создаст файл, который можно открыть в Figma со всеми слоями, шрифтами и отступами.

### 3. Интеграция в твой GitHub репозиторий
Так как твой проект `training_jump` использует современные веб-технологии, вот краткий план интеграции:
1. **Шрифты:** Я использовал шрифт `Lexend`. Убедись, что он подключен в твоем проекте (через Google Fonts или локально).
2. **Стили (Tailwind):** Дизайн построен на Tailwind CSS. Если ты его не используешь, тебе нужно будет либо установить его, либо адаптировать стили под свой CSS/SCSS.
3. **Компоненты:** Ты можешь копировать структуру блоков (карточки с матовым стеклом, кнопки с градиентами) и создавать из них переиспользуемые компоненты в своем коде.
4. **Анимации:** Для эффектов 3D и появления я рекомендую использовать библиотеку `framer-motion` (если у тебя React) или просто CSS Transitions, которые я заложил в код.
